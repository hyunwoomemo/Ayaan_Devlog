# passport 모듈을 이용한 인증 앱 생성하기

Passport를 이용해서 만들 앱은 이메일과 비밀번호를 이용해서 로그인하는 것과 구글 아이디를 이용하여 OAuth 로그인을 할 수 있는 앱을 구현

passport는 `세션`을 이용함

```bash
npm install dotenv express nodemon body-parser cookie-parser cors mongoose passport 
```
